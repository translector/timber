Theme Information
-------------------------------------------------------------------------------
Timber Drupal 8
- Theme Version: 8.x-5.3
- Released on: May 11, 2018
- Compatible with Drupal 8.x
- Author: Arun Kumar

Installation
-------------------------------------------------------------------------------
From Demo package:
-	Extract download package to your host
-	Import file README_DATABASE/timber.sql to your database
-	Go to folders: web/sites/default
-	Rename _settings.php to settings.php
-	Open file settings.php and change lines: 788, 789, 790 to your database information
-	Run website by browser and you can start to try Timber and Drupal 8
